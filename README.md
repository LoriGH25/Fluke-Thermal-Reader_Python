# Fluke Reader Python

Progetto Python per la lettura di dati da dispositivi Fluke.

## Descrizione

Questo progetto permette di interfacciarsi con dispositivi di misurazione Fluke per la lettura e l'elaborazione di dati.

## Installazione

```bash
pip install -r requirements.txt
```

## Utilizzo

```python
# Esempio di utilizzo
from fluke_reader import FlukeReader

reader = FlukeReader()
data = reader.read_measurements()
```
